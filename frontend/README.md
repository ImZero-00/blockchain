# E-commerce Blockchain - Frontend

React frontend với Vite, React Router.

## Setup

```powershell
# Cài đặt
npm install

# Chạy dev server
npm run dev

# Build cho production
npm run build
```

## Pages

- `/` - Danh sách sản phẩm
- `/checkout` - Trang đặt hàng
- `/verify` - Xác minh đơn hàng

Frontend chạy tại: `http://localhost:3000`
