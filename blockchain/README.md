# E-commerce Blockchain - Blockchain Module

Smart Contract Solidity cho việc xác thực giao dịch ecommerce trên blockchain.

## Chạy lệnh

```bash
# Cài đặt
npm install

# Compile contract
npx hardhat compile

# Test
npx hardhat test

# Chạy local node
npx hardhat node

# Deploy (terminal mới)
npx hardhat run scripts/deploy.js --network localhost
```

## Contract Address

Sau khi deploy, contract address sẽ được lưu trong `deployed-address.json`.
